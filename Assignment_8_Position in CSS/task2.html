Absolute Relative
syntax: {position: absolute} syntax: {position: relative}
It is positioned relative to its nearest positioned (not static) ancestor, or to the initial containing block if no positioned ancestor is found. It is positioned relative to its normal position in the document flow.
The element can be precisely placed using top, right, bottom, or left properties. The element is then offset by values given for top, right, bottom, or left properties, which move the element from its initial position.
Absolute positioning removes the element from the normal document flow, and it doesn't affect the layout of other elements. Other elements in the document flow are not affected by the positioned element, and the space it originally occupied is reserved.
If no positioned ancestor is found, the element is positioned relative to the initial containing block (usually the <html> element). If no offset values are provided, the element remains in its normal position.
